## 🔑 如何获取 API Keys

### Anthropic API Key
1. 访问 https://console.anthropic.com/
2. 登录账号
3. 点击 **"API Keys"**
4. 点击 **"Create Key"**
5. 复制密钥（格式：`sk-ant-api03-...`）

### Google API Key
1. 访问 https://makersuite.google.com/app/apikey
2. 登录 Google 账号
3. 点击 **"Create API key"**
4. 复制密钥（格式：`AIzaSy...`）

## 🎨 快速定位图标

**环境变量入口可能的图标：**

| 图标 | 名称 | 位置 |
|------|------|------|
| 🔑 | 钥匙 | 左侧导航栏 "Variables" |
| ⚙️ | 齿轮 | 右上角 "Settings" |
| + | 加号 | 添加新变量 |
| 🔒 | 锁 | 加密变量值 |

---

## 💡 提示

1. **加密敏感信息** - 勾选 "Encrypt" 保护 API 密钥
2. **变量名称全大写** - 如 `ANTHROPIC_API_KEY`
3. **不要加引号** - Value 部分直接粘贴，不要加引号
4. **保存后自动重新部署** - 修改变量后会自动触发部署

## 🆘 仍然找不到？

### 替代方法：在部署配置页面

1. 点击项目
2. 进入 **"Settings"** 标签
3. 滚动到 **"Environment Variables"** 部分
4. 点击 **"New Variable"**

### 或者：在新建项目时

在 **"Deploy from GitHub"** 页面：
- 向下滚动找到 **"Environment Variables"** 部分
- 点击 **"Add Variable"**
- 添加完成后点击 **"Deploy Now"**
